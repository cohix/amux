# Release v0.1.0

Initial release of `aspec-cli` — a CLI tool for managing agentic software projects with spec-driven development.

## Features

- **TUI mode**: Interactive terminal UI for managing and running agentic workflows
- **CLI commands**:
  - `ready` — Prepare a development environment with Docker, with `--build` and `--no-cache` flags
  - `new` — Scaffold new work items from templates
  - `implement` — Run code agent to implement work items, with `--plan` flag for plan-only mode
  - `chat` — Interactive chat with the code agent, with `--plan` flag support
- **Docker-based execution**: All agentic operations run inside Docker containers for safety

## Fixes

- Fixed Docker build log streaming in TUI mode

## Notes

- Single statically-linked binary for macOS, Linux, and Windows
- Per-repo configuration via `aspec/.aspec-cli.json`
- Global configuration via `$HOME/.aspec/config.json`
